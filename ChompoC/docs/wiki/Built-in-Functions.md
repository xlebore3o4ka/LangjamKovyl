# Встроенные функции

Все функции ниже являются callable-значениями. Их можно присваивать переменным и передавать в пользовательские функции. `print(...)` является отдельной инструкцией языка.

## Преобразования

### `Int(value)`

Возвращает `integer`.

- `bool`, `integer`, `char` преобразуются к целому;
- `double` усекается к нулю после проверки диапазона;
- строка должна целиком содержать корректное целое;
- `NULL` преобразуется в `0`.

```javascript
Int("42"); // 42
Int(3.9);  // 3
Int('A');  // 65
```

### `Double(value)`

Принимает число, `bool`, `char`, числовую строку или `NULL`. Для `NULL` возвращает `0.0`.

### `Bool(value)`

Возвращает truthiness значения.

### `String(value)`

Возвращает строковое представление. Callable преобразовывать через `String` нельзя.

### `Char(value)`

- `char` возвращается без изменения;
- строка должна иметь длину ровно один байт;
- integer должен находиться в диапазоне `0..255`.

### `Array(value)`

- массив возвращается как тот же ссылочный объект;
- строка преобразуется в массив `char`;
- другое значение оборачивается в одноэлементный массив.

### `CATS(array)`

`Char Array To String`: преобразует массив, состоящий только из `char`, в строку.

### `Type(value)`

Возвращает одно из имён:

```text
NULL
bool
integer
double
char
string
array
callable
```

## Последовательности

### `len(sequence)`

Принимает строку или массив. Возвращает число байтов или элементов.

### `push(array, values...)`

Добавляет один или несколько элементов и возвращает новую длину. Добавление прямого или косвенного цикла массивов запрещено.

### `pop(array)`

Удаляет и возвращает последний элемент. Для пустого массива возвращает `NULL`.

### `removeAt(array, index)`

Удаляет и возвращает элемент по индексу. Индекс обязан существовать.

```javascript
var values = Array{10, 20, 30};
print(removeAt(values, 1)); // 20
print(values);              // {10, 30}
```

## Системные функции

### `args()`

Возвращает новый массив строк с аргументами после имени `.chmp`-файла.

```bash
Chompo program.chmp host 4040
```

```javascript
args(); // {host, 4040} — оба элемента имеют тип string
```

## Ввод и вывод

### `input()`

Блокирующе читает одну строку. Возвращает `NULL` на EOF.

### `inputPoll(timeoutMs = 0)`

Возвращает:

```javascript
Array{"data", line}
Array{"wait"}
Array{"closed"}
```

`timeoutMs` равен `0`, положительному числу миллисекунд или `-1` для неограниченного ожидания. Для интерактивной консоли готовностью считается завершённая строка.

### `flush()`

Сбрасывает текущий выходной поток и возвращает `NULL`.

### `istream(path = "standart")`

Переключает входной поток.

### `ostream(path = "standart", mode = "rewrite")`

Переключает поток `print`. Режимы: `rewrite`, `append`, `create`.

### `iostream(inputPath = "standart", outputPath = "standart", mode = "rewrite")`

Атомарно подготавливает оба файловых потока и переключает их.

## TCP Network API

### `netListen(host, port, backlog = 16)`

Создаёт listener. Порт `0` разрешает ОС выбрать свободный порт.

### `netConnect(host, port)`

Подключается и возвращает socket handle.

### `netAccept(listener)`

Возвращает socket handle или `NULL`, когда очередь подключений пуста.

### `netPoll(handles, timeoutMs = 0)`

Возвращает массив handles, готовых для accept/read/обработки закрытия.

### `netSend(socket, data)`

Низкоуровневая неблокирующая отправка. Возвращает фактически отправленное число байтов, которое может быть меньше длины строки.

### `netSendAll(socket, data, timeoutMs = 5000)`

Пытается отправить всю строку и возвращает статус как данные:

```javascript
Array{"sent", bytes}
Array{"timeout", bytes}
Array{"error", bytes, message}
```

`timeoutMs = -1` отключает лимит. Функция может удерживать однопоточный интерпретатор до завершения, timeout или ошибки.

### `netReceive(socket, maxBytes = 4096)`

Максимум — 1 MiB. Возвращает:

```javascript
Array{"data", chunk}
Array{"wait"}
Array{"closed"}
```

Системная socket-ошибка становится runtime-ошибкой.

### `netReceiveLine(socket)`

Читает до `\n`, удаляя `\n` и завершающий `\r`. Возвращает:

```javascript
Array{"data", line}
Array{"wait"}
Array{"closed"}
Array{"error", message}
```

Внутренний лимит одной строки — 1 MiB. Ошибки соединения возвращаются как статус, чтобы сервер мог удалить одного клиента и продолжить работу.

### `netPort(handle)`

Возвращает локальный TCP-порт.

### `netClose(handle)`

Закрывает listener/socket и возвращает `NULL`. Повторное использование handle является ошибкой.

## `print(arguments...)`

`print` вычисляет аргументы слева направо и выводит их подряд. Пробелы и `\n` нужно передавать явно.

```javascript
print("x=", 10, "\n");
flush();
```
