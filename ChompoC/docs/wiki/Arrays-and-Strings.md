# Массивы и строки

## Массивы

Массив создаётся через `Array{...}`:

```javascript
var values = Array{1, "two", true};
var empty = Array{};
```

Элементы могут иметь разные runtime-типы.

## Ссылочная семантика

Массив является изменяемым ссылочным объектом:

```javascript
var first = Array{1, 2};
var alias = first;

alias[0] = 10;
print(first); // {10, 2}
```

Присваивание массива не копирует его элементы. Конкатенация и повторение создают новый массив.

## Индексация и вложенная мутация

```javascript
var matrix = Array{Array{1, 2}, Array{3, 4}};
print(matrix[1][0]); // 3
matrix[0][1] = 20;
```

Индекс должен быть integer-like значением и находиться в диапазоне `0..len(array)-1`. Отрицательные индексы не поддерживаются.

## `push`, `pop`, `removeAt`

```javascript
var values = Array{};

push(values, 10);
push(values, 20, 30);

print(pop(values));         // 30
print(removeAt(values, 0)); // 10
print(values);              // {20}
```

- `push(array, values...)` добавляет минимум один элемент и возвращает новую длину;
- `pop(array)` удаляет последний элемент, пустой массив даёт `NULL`;
- `removeAt(array, index)` удаляет и возвращает элемент по индексу.

`push` имеет амортизированную сложность O(1) на одиночное добавление. `removeAt` имеет O(n), потому что последующие элементы сдвигаются.

## Конкатенация и повторение

```javascript
var joined = Array{1, 2} + Array{3}; // {1, 2, 3}
var repeated = Array{'a', 'b'} * 2; // {a, b, a, b}
```

Обе операции создают новый внешний массив. Вложенные массивы остаются теми же ссылочными объектами.

## Циклические ссылки

Прямые и косвенные циклы запрещены:

```javascript
var value = Array{};
push(value, value); // runtime-ошибка
```

Проверка выполняется перед изменением, поэтому неудачный `push` не оставляет массив в частично изменённом состоянии.

## `len` и `in`

```javascript
len(Array{1, 2, 3}); // 3
2 in Array{1, 2, 3}; // true
```

Membership использует обычное равенство Chompo, включая глубокое сравнение вложенных массивов.

## `for-in` и snapshot

```javascript
var values = Array{1, 2, 3};

for (var value in values) {
    push(values, 4);
    print(value, "\n");
}
```

Цикл обходит snapshot, созданный перед первой итерацией. Изменение исходного массива не меняет текущий набор итераций.

## Строки

Строка хранит последовательность байтов:

```javascript
var text = "Chompo";
print(text[0]); // C
text[0] = 'c';
```

Индексирование возвращает `char`. Для записи в строку требуется `char`.

Присваивание строки создаёт независимое значение: изменение индексированного символа одной переменной не меняет другую переменную.

## Строковые операции

```javascript
"hello " + 42       // "hello 42"
"ab" * 3            // "ababab"
len("abc")          // 3
'a' in "chat"       // true
"ha" in "chat"     // true
```

Строки не являются Unicode-aware: длина, индекс и лимиты чата измеряются в байтах.

## Преобразования строк и char-массивов

```javascript
Array("abc")               // {a, b, c}
CATS(Array{'o', 'k'})       // "ok"
Char(65)                    // 'A'
```

`CATS` требует массив, каждый элемент которого имеет тип `char`.
