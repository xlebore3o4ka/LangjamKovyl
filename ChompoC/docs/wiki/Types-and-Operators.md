# Типы и операторы

## Runtime-типы

| Тип | Пример | Особенности |
|---|---|---|
| `NULL` | `NULL` | отсутствие значения |
| `bool` | `true` | участвует в числовых операциях как `1`/`0` |
| `integer` | `42` | знаковое 64-битное целое |
| `double` | `3.14` | число с плавающей точкой |
| `char` | `'A'` | один байт, числовое значение байта |
| `string` | `"text"` | последовательность байтов со значимой копией |
| `array` | `Array{1, 2}` | изменяемый ссылочный объект |
| `callable` | `someFunction` | native- или пользовательская функция |

## Truthiness

Ложными являются:

- `NULL`;
- `false`;
- числовой ноль;
- `\0`;
- пустая строка;
- пустой или отсутствующий массив.

Остальные значения истинны. Callable всегда истинны.

## Арифметика

```javascript
1 + 2
5 - 3
4 * 6
7 / 2
7 % 2
```

`bool` и `char` считаются integer-like числами. Если один числовой операнд является `double`, результат арифметики вычисляется как `double`. Деление двух integer-like значений является целочисленным. Деление и остаток от нуля вызывают runtime-ошибку.

Дополнительная перегрузка:

```javascript
"value=" + 10             // "value=10"
Array{1} + Array{2, 3}    // новый массив {1, 2, 3}
"ab" * 3                  // "ababab"
Array{1, 2} * 2           // новый массив {1, 2, 1, 2}
```

Отрицательное количество повторений запрещено.

## Сравнение и равенство

```javascript
left < right
left <= right
left > right
left >= right
left == right
left != right
```

Порядковые сравнения работают для числовых значений. `integer`, `double`, `bool` и `char` сравниваются численно.

Массивы сравниваются глубоко по элементам. Две ссылки на один массив равны сразу. Callable равны, только когда указывают на тот же callable-объект.

## Логика

```javascript
!value
left && right
left || right
```

`&&` и `||` используют short-circuit evaluation и возвращают `bool`, а не один из исходных операндов.

## Membership

```javascript
needle in array
'a' in "chat"
"ha" in "chat"
```

Для массива используется равенство элементов. В строке можно искать `char` или подстроку.

## Присваивания и обновления

```javascript
value = result;
value += amount;
value -= amount;
value *= amount;
value /= amount;
++value;
value--;
```

Присваивания правоассоциативны. Compound-операторы используют те же правила, что соответствующие бинарные операции. Обновление `char` обязано оставаться в диапазоне `0..255`.

## Приоритет операторов

От низкого к высокому:

1. `=`, `+=`, `-=`, `*=`, `/=`;
2. `||`;
3. `&&`;
4. `==`, `!=`;
5. `<`, `<=`, `>`, `>=`, `in`;
6. `+`, `-`;
7. `*`, `/`, `%`;
8. prefix `!`, `-`, `++`, `--`;
9. вызов `()`, индекс `[]`, postfix `++`, `--`.

Скобки изменяют порядок вычисления.

## Строковое представление

`print` и `String` используют runtime-представление:

```text
NULL
true / false
42
3.14
text
{1, 2, 3}
<function name>
```

`String(callable)` запрещён, хотя `print` может вывести callable через общее runtime-представление.
