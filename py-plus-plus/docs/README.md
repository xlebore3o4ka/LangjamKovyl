# Py++ Documentation

This documentation set is organized into three main sections:

- `docs/syntax/` — language syntax and grammar patterns.
- `docs/semantics/` — runtime behavior, evaluation rules, and execution semantics.
- `docs/examples/` — annotated example programs that demonstrate real usage.

Each section is stored in its own folder and includes subheadings, explanations, and links to concrete example files in `examples/`.
