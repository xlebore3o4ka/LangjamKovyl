# Py++ Language Documentation

This documentation is organized into separate markdown sections:

- `syntax.md` — syntax rules and core language constructs.
- `semantics.md` — language semantics and runtime behavior.
- `usage.md` — example categories and how to run example programs.

## Examples folder layout

Examples are stored in the repository `examples/` folder with the following categories:

- `examples/syntax/` — syntax-focused examples.
- `examples/semantics/` — semantics and behavior examples.
- `examples/macros/` — macro examples and compile-time expansion.
- `examples/imports/` — import and module examples.

Each example file includes comments that explain the purpose and behavior of the code.
